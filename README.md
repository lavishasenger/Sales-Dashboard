# Sales-Dashboard Project

Process
* Cleaned a raw dataset of 1,026 rows to eliminate 26 duplicates and standardized shorthand entries such as marital status and gender.
* Transformed continuous raw age values into clean target segments (*Adolescent, Middle Age, Old*) using nested `IF` logic.
* Applied advanced formulas like `XLOOKUP` and `AVERAGEIFS` to run multi-variable calculations across demographics and income arrays.
* Constructed multi-dimensional Pivot Tables that uncovered a peak 58.9% conversion rate in the Pacific region and high intent among professionals.
* Mapped data aggregations into dynamic charts synchronized via interactive Slicers to enable real-time dashboard filtering for stakeholders.

 Key Insight
 * Middle-aged professionals with a Bachelor's degree represent the primary purchasing demographic for the business.
 * Higher average income ($\approx \$55,000\text{--}\$60,000$) and short commute distances ($0\text{--}1$ miles) are the strongest drivers for a bike purchase.
 * Single individuals and customers in the Pacific region exhibit the highest conversion efficiency, whereas married couples and long-distance commuters are the       least likely to buy.
